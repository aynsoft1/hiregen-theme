# Hiregen Recruitment WordPress Theme

HiREGen is a modern, responsive, and performance-optimized WordPress theme designed specifically for recruitment agencies, job boards, and HR consultancies. Built with **Bootstrap 5**, it offers a sleek design with robust functionality for managing job listings (requires WP Job Manager).

## ğŸ“¦ Installation

1.  **Download:** Get the `hiregen-recruitment.zip` file.
2.  **Upload:** Go to your WordPress Dashboard -> **Appearance** -> **Themes** -> **Add New** -> **Upload Theme**.
3.  **Activate:** Select the file and click **Install Now**, then **Activate**.
4.  **Plugins:** The theme works best with the **Hiregen CPTs** plugin (for custom post types) and **WP Job Manager** (for job functionalities).
